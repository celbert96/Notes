# Working with Sessions

### Create a new session
```tmux new -s {session-name}```

### List Sessions
```tmux ls```

### Detatch a Session
```tmux detach```

### Attach a Session
```tmux attach -t {session-name}```

### Reload Plugins
```{prefix} shift+i```

### Create New Window
```{prefix} c```

### Rename Window
```{prefix} ,```

### Switch to Window By Num
```{prefix} {num}```

### Switch to Next/Prev Window
```{prefix} {n/p}```

### View Windows
```{prefix} w```
